## ℹ️ Overview

That's my first basic python project, from an idea to working thing...
It's terminal application working as a reminder of daily sale on x-kom.pl website.


## ⬇️ Installation

Packages to install:

```bash
pip install schedule
pip install beautifulsoup4
```


## 🚀 Usage

Run following command on the terminal to start the app:

```py
python3 main.py &
```

with '&' to work in the background
